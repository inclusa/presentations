Tipos de clientes

Clientes ligeros


Los clientes ligeros son básicamente el equipo donde se conectan teclado, ratón y monitor ya que no ejecutan nada.

  • Ventajas

    Sin mantenimiento ya que no tiene nada instalado
    Administración centralizada
    Menor consumo energético

  • Uso recomendado

    Aplicaciones ofimáticas, JClic,...
    Navegar

  • Uso no recomendado

    Virtualización
    Edición de video
    Usos que requieran mucho uso de disco o procesador

Nota: Las aplicaciones se ejecutan en el SERVIDOR.

Clientes semi-ligeros


Los clientes semi-ligeros son equipos normales sin disco duro

  • Ventajas

    No requieren mantenimiento ya que no tiene nada instalado
    Administración centralizada
    Menor consumo energético

  • Uso recomendado

    Equipos a los que se conecte hardware específico (PDIs)
    Equipos con un mínimo de 2Gb de ram
    Entornos donde el servidor no tenga muchos recursos

  • Uso no recomendado

    Virtualización
    Edición de video
    Usos que requieran mucho uso de disco o procesador

Nota: Las aplicaciones se ejecutan en el CLIENTE.

Clientes pesados


Son equipos normales con todos sus componentes

  • Ventajas

    Se aprovechan todos los recursos hardware del cliente
    Cada equipo puede tener su propio software
    Tienen más posibilidades de ampliar periféricos

  • Uso recomendado

    Equipos nuevos
    Equipos a los que se conecte hardware específico (PDIs)
    Entornos donde se quierea virtualizar, editar video, etc.

  • Uso no recomendado


    Equipos viejos o con pocos recursos

Nota: Las aplicaciones se ejecutan en el CLIENTE.

Raspberry-Pi


Son equipos baratos (40€ ) funcionan más o menos como clientes ligeros

  • Ventajas

    Son muy baratos
    No requieren mantenimiento
    Ocupan muy poco espacio

  • Uso recomendado

    Ordenadores de consulta (Biblioteca, hall, etc.)
    Aulas de informática con uso poco exigente
    Si se dispone de un servidor actualizado

  • Uso no recomendado


    Lo mismo que para los clientes ligeros

Nota:No confundir con Arduino o Raspberry-zero.
