Por qué utilizar Software Libre

El Software Libre persigue 4 libertades

  • Libertad #0: USAR

    Libertad de usar el programa para cualquier propósito.

  • Libertad #1: ESTUDIAR

    Libertad para estudiar cómo funciona el programa y modificarlo.

  • Libertad #2: DISTRIBUIR

    Libertad para distribuir cópias del programa.

  • Libertad #3: MEJORAR

    Libertad para mejorar el programa y hacer públicas las copias.
